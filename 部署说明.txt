# 记账网站部署指南

## 文件清单
请确保上传以下4个文件到GitHub仓库：
✅ index.html - 网站主页面
✅ style.css - 样式文件
✅ script.js - 功能脚本
✅ README.md - 说明文档

## GitHub Pages 部署步骤

### 1. 创建GitHub账号
访问：https://github.com
注册账号并登录

### 2. 创建新仓库
- 点击右上角 "+" → "New repository"
- 仓库名：my-expense-tracker
- 设置为 Public（公开）
- 勾选 "Add a README file"
- 点击 "Create repository"

### 3. 上传文件
- 点击 "uploading an existing file"
- 拖拽上传4个文件
- 提交信息：Initial commit - 记账网站
- 点击 "Commit changes"

### 4. 启用GitHub Pages
- 进入仓库 Settings
- 找到 Pages 设置
- Source 选择 "Deploy from a branch"
- 分支选择 "main"
- 点击 "Save"

### 5. 获取访问链接
等待几分钟后，访问链接会显示为：
https://你的用户名.github.io/my-expense-tracker/

## 其他快速部署方案

### Netlify（最简单）
1. 访问：https://netlify.com
2. 拖拽整个文件夹到页面
3. 自动获得访问链接

### Vercel
1. 访问：https://vercel.com
2. 连接GitHub仓库
3. 自动部署

## 注意事项
- 部署后网址是公开的，任何人都可以访问
- 数据依然存储在各自设备的浏览器中
- 使用"备份数据"和"恢复数据"功能进行设备同步

## 访问效果
部署成功后，你可以：
✅ 在任何设备的浏览器中输入网址访问
✅ 收藏网址到浏览器书签
✅ 添加到手机桌面，像APP一样使用
✅ 分享给家人朋友使用
